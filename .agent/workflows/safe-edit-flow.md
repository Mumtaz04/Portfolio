# Safe Edit Flow

1. **Read Rules**:

   - Read \.agent/rules/safe-mode-rules.md\ to ensure you are following the safety protocols.

2. **Understand Context**:

   - Identify the file(s) to be changed.
   - Check for dependencies and usages of the code to be changed.

3. **Plan Changes**:

   - Create a plan for surgical edits.
   - Limit scope to only what is requested.

4. **Apply Edits**:

   - Apply changes carefully.
   - Verify syntax and structure.

5. **Verify**:
   - Check if the changes affect other files.
   - Run available tests or lint checks.
